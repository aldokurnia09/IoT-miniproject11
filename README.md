# IoT-Mini Project 11

ESP8266 dihubungkan dengan MQTT Dash untuk mengontrol Stepper Motor

--------------------------------------------------

### Hardware yang digunakan: 
- NodeMCU ESP8266
- Kabel Jumper
- Project Board
- Stepper Motor 28BYJ-48
- Stepper Motor Driver Board

### Software yang digunakan: 
- Arduino IDE
- MQTT Dash
